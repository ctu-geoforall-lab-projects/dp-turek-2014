The file "cam_distortion.diff" includes modifications in GRASS "i.ortho.camera" module to extend its data model for support of distortion. 

Another file "vrectify_to_lib.diff" includes changes in GRASS which were made by moving Helmert transformation code from v.rectify module into GRASS Imagery Library. 

The directory "i.ortho.bba" includes prototype which was tested on data in directory "data" which includes also protocol of Bingo-F adjustent.
